# Замена схожих русских символов на английские аналоги

## Описание

Начертание различных букв зависит от шрифта, которым напечатан текст. Всё ниженаписанное справедливо для наиболее распространённых шрифтов:
1. Courier New
2. Times New Roman
3. Arial
4. Roboto
5. Helvetica Neue

И некоторых других.
